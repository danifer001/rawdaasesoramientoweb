# Raw DA - Asesoramiento Web UX

Sitio web profesional de asesoramiento UX/UI con enfoque brutalista y transparente. Detección de problemas web, optimización de conversiones y capacitación en UX real.

## 🎯 Objetivo

Landing page que comunica servicios de asesoramiento web con total honestidad, sin filtros ni promesas vacías. Diseño brutalista que refleja el enfoque directo y técnico del servicio.

## 🚀 Características

- **Diseño Brutalista**: Estética técnica con colores negro, blanco y verde
- **Tipografía Monoespaciada**: Fuentes Space Mono y Roboto Mono
- **Scroll Snapping**: Navegación fluida entre secciones
- **Responsive**: Adaptado para mobile y desktop
- **Formulario de Contacto**: Integración directa para diagnósticos
- **WhatsApp Flotante**: Comunicación inmediata

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 con variables personalizadas
- JavaScript vanilla
- Google Fonts (Space Mono, Roboto Mono)
- Font Awesome (iconos)
